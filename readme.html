Téléchargez le fichier .zip depuis le dépôt GitHub. Extrayez l'archive et renommez le dossier extrait en "ecommerce". Une fois le dossier "ecommerce" ouvert, renommez le sous-dossier en "wordpress" pour que le lien du site devienne : http://localhost/ecommerce/wordpress. Créez une base de données nommée "ecommerce_db" dans votre serveur MySQL. La base de données requise pour le projet se trouve également dans ce dépôt GitHub sous le nom "ecommerce_db (1).sql", aux côtés des fichiers du projet. Importez-la dans votre serveur.
